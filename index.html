Your Projects

Music Library

Music library where the user can store the information of the albums they own or want. The user is then able to filter the database according to several parameters (artist, release year, ownership, and format) using advanced patterns (suggestions are included in the filter text input). Any individual entry can be deleted manually.

Web-based Calculator

Fully working calculator able to sum, subtract, multiply and divide any two decimal numbers. The user is also able to carry the result of an operation onto the next. A filesafe is also in place when attempting to divide by zero. The calculator includes a function to correct the input at any point by deleting characters. It also includes full keyboard support.

Improved Etch-A-Sketch

Interactive web-based Etch-A-Sketch where you can draw while holding any mouse button. It includes the possibility to modify the density of the grid, i.e. the number of drawing squares (1–100). Additionaly it includes three different paint modes.
